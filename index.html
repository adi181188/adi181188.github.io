Koloder Privacy Policy
This privacy policy governs your use of the software application Koloder(“Application”) for mobile
devices that was created by adi aharon at no cost and is intended for use as is. The Application helps
users to accomplish everyday tasks hands free using voice recognition with the exception of the setting
screen.
Consent
By downloading, installing and using the Application you agree to the terms and conditions set forth in
this Privacy Policy. If you disagree you may uninstall the Application from your mobile device.
What information does the Application obtain
Automatically Collected Information
We do not collect any of your personal data, however we use or may use third parties software like
crash management software, analytics software or advertising network software that may collect
certain anonymous information like the way you use the Application, your mobile operating system,
type of mobile device, app logs, device language, whether the device is rooted.
Collected Information that is required for use of the application and
how it is used
1. google account: we do not collect or use the google account information other than for the purpose
of in app account management using firebase api meaning the user will only use the google account in
order to login to the app using firebase.
2.gmail account: we do not use or collect the gmail account information other than for the purpose of
providing the user with the ability to send an email from the app using gmail api. we do not store or
share any of the account information outside of the application.
3.phone contacts: the information is used in order to provide the user with the ability to make calls,
send messages, emails to a contact in the phone. the contacts are stored in the application in order to
provide the user with fast and reliable use of the application. we do not store or share any of the
contacts with a server or anywhere outside of the application other then for the use of google speech
recognizer and gmail api.
Why does the Application need access to my microphone?
the application uses the pocket sphinx library and google speech recognizer to provide the user with the
ability to use the application hands free(with the exception of the setting screen). the app records the
voice and uses it only with these library’s we do not store or share the records outside of the
application.
Why does the application need access to my contacts?
the contacts are required in order to provide the user with the ability to use actions within the
application and in order to make the application usage simpler for the user. we do not share or send the
contacts information anywhere and it's only available for the user inside the application.
